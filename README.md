#### Install
You need Linux distribution like Ubuntu or Manjaoro

```sh
https://github.com/bensuperpc/docker-ShivyC.git
```
```sh
cd docker-ShivyC
```
#### Usage

```sh
./compiler.sh make (where project to build)
```
#### Build
```sh
make linux/amd64
```
